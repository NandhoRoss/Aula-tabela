Tabela

para criar uma tabela vamos utilizar a tag table para iniciar o processo da tabela, dentro dela que estarão todos os elementos de uma tabela

dentro então vamos começar a representação do cabeçalho das colunas desta tabela que será feita a partir do thead

dentro da tag thead teremos a tag tr que significa que ela será representada em linha.

entro a tag tr teremos a tag th que será a quantidade de colunas com seus respectivos nomes.

feita a parte do cabeçalho iremos para parte do corpo da tabela que conterá os demais dados, e para criar este corpo usaremos a tag tbody

dentro do tbody entraremos com os dados da tabela também em linha utilizando mais uma vez a tag tr para colocar os dados dentro das linhas, mas agora ao invés de usar o th usaremos o td para o preenchimento dos dados na coluna.

assim é feito o preenchimento da tabelas, mas para auxiliar na formatação da tabela temos alguns atributos que podemos colocar dentro da tag td que são o colspan="x" e rowspan="x" que fazem que uma coluna assuma o espaço de 2 ou mais colunas e uma linha assuma o espaço de 2 ou mais linhas respectivamente

e para finalizar nossa tabela temos o tfoot onde serão colocado nosso dados importante como valores totais e resumos de nossa tabela. A estrutura será como o tbody onde teremos uma tr e dentro quantas tds forem necessárias.
